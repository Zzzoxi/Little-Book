# 数据结构与算法
1. 算法的特性：
输入、输出、有穷性、确定性、可行性
2. 算法设计的要求：

    1. 正确性（算法程序对于非法的输入结构能够得出满足规格说明的结果）
    2. 可读性
    3. 健壮性（输入数据不合法时，算法也能做出相关处理）
    4. 时间效率低和存储量低
## 2.9算法时间复杂度
O()

O(1) < O(logn) < O(n) < O(nlogn) < O(n*n)

2.9.2推导大O阶方法
1. 用常数1取代运行时间中的所有加法常数
2. 在修改后的运行次数函数中，只保留最高阶项
3. 如果最高阶项存在且不是1，则去除与这个项相乘的常数
